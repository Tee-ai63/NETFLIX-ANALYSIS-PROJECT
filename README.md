Netflix Content Strategy Analysis
This project analyzes Netflix's global content strategy using data visualization. The research examines content distribution, genre performance, and regional catalog optimization.

Key Findings:

Content portfolio skews toward TV-MA (35%) with strong US dominance (46%)

International thrillers and teen dramas identified as high-opportunity gaps

Documentaries show 3x longer shelf life than comedies

Southeast Asia shows underserved family/kids content potential

The dashboard reveals Netflix's "global first, locally optimized" approach, balancing universal hits with targeted regional investments.

Data Source: Netflix Content Library
Tools: Tableau Public
